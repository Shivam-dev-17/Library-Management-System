# Library Management System using Java
# Overview
Asimple Library Management System implemented in Java using SQLite(JDBC).Supports book/user CRUD, issue/return workflow and reports.
# Features
- Add / update / list books
- Register / list users
- Issue / return books (with due date)
- Overdue report

# Tech Stack 
- Java 17
- Maven
- SQLite(embedded via sqlite-jdbc)

# Setup & Run
1. Clone repo
2. Build:
   ```bash
   mvn clean package
